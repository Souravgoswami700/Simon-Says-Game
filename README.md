🎮 Game Overview
Simon Says is a classic memory and reaction game where players must repeat increasingly longer sequences of colored button presses. The game tests your memory, timing, and focus as the sequence grows longer with each successful round.

✨ Features
.4 colored buttons (Red, Green, Blue, Yellow) with distinct sounds

.Progressive difficulty - sequence length increases each round

.Visual & audio feedback for correct/incorrect presses

.Score tracking and high score persistence

.Start/Strict mode toggle

.Responsive design works on desktop & mobile

📱 How to Play
.Click START (or press any colored button) to begin

.Watch & Listen as Simon plays a sequence of colors

.Repeat the exact sequence by clicking the colored buttons

.Continue as the sequence gets longer each round

.Lose if you press wrong button or out of sequence

.Win by reaching 20+ rounds (expert level!)
